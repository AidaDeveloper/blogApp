=== Инструкция по развёртыванию проекта BlogApp ===

Стек проекта:
- Backend: PHP, MySQL
- Frontend: Vue.js с TailwindCSS

1. Настройка базы данных:
   - Создать базу MySQL с именем `blog_app`.
   - Импортировать дамп базы
   - Конфигурация подключения находится в файле `config.php`.

2. Запуск локального веб-сервера (Backend):
   - Перейти в папку backend проекта:
     cd backend
   - Запустить сервер:
     php -S localhost:8000

3. Запуск фронтенда (Vue.js + Tailwind):
   - Перейти в папку фронтенда:
     cd frontend
   - Установить зависимости:
     npm install
   - Запустить dev-сервер:
     npm run dev
   - Открыть проект в браузере по ссылке
